# Go + AI 实习生学习手册

> 欢迎加入团队。
>
> 请fork这个项目，然后参考本手册开始你的编程之旅。
>
> 这不是一份“有人一步步教你”的实习计划，而是一套**帮助你在 AI 时代真正成长为工程师的训练系统**。
>
> 在这里，我们不以“学了多少”为标准，只看一件事：**你最终能独立交付什么。**

---

## 一、我们希望你成为什么样的实习生

在实习结束时，你不需要：
- 精通所有 Go 语法
- 背下复杂算法
- 记住所有框架细节

但你**必须能够**：
- 使用 **Go 语言 + AI 编程工具** 独立完成小型项目
- 清楚地解释：你在做什么、为什么这么做
- 面对新需求，知道如何拆解问题并推进

一句话总结：
> **你是“会使用 AI 的工程师”，而不是“被 AI 带着走的人”。**

---

## 二、基本学习与工作方式

### 1. 默认使用 AI 编程

在这里：
- 使用 Cursor / Claude / ChatGPT **不是作弊**，而是基本能力
- 不使用 AI，反而会被认为效率不足

但请注意：
- ❌ 直接复制 AI 代码却不知道在干什么
- ✅ 能说清楚：哪些是 AI 写的、你改了什么、为什么改

我们更看重你**如何与 AI 协作**。

---

### 2. 学习 = 产出

没有“我还在学”的状态。

每一个阶段，都必须有：
- 能运行的代码
- 清晰的 README
- 你自己的理解说明

---

### 3. 我们的评价标准

我们只关注三件事：
1. **你交付了什么**
2. **你是否能解释清楚**
3. **你是否在持续进步**

---

## 三、整体学习路径（8–10 周）

| 阶段 | 时间 | 目标 |
|---|---|---|
| Phase 0 | 3 天 | 工具与 AI 编程入门 |
| Phase 1 | 1–2 周 | Go 基础 + 小程序 |
| Phase 2 | 2 周 | Go Web 服务 |
| Phase 3 | 2 周 | 工具型小项目 |
| Phase 4 | 2–3 周 | 业务型小项目 |

---

## 四、Phase 0：工具与 AI 编程入门（3 天）

### 目标
- 熟悉基本工程工具
- 学会“如何向 AI 提问”

### 必备工具
- Cursor（主力 IDE）
- Claude / ChatGPT
- Git & GitHub

### 你需要完成
- 用 AI 解释一段 Go 代码
- 用 AI 重构一段代码
- 独立创建一个 Go 项目并运行

### 交付物
- 一个可运行的 Go Hello World 项目
- 一篇短文：《我如何用 AI 帮我写代码》

---

## 五、Phase 1：Go 基础（1–2 周）

### 目标
- 能看懂代码
- 能修改代码
- 能补全简单功能

### 学习重点
- Go 基本语法
- struct / interface
- 错误处理
- 常用标准库

### 练习项目（任选其一）
- Todo 命令行工具
- 简单日志分析器
- 小型爬虫工具

### 交付物
- 1 个 CLI 项目
- README（功能 + 使用说明 + AI 使用说明）

---

## 六、Phase 2：Go Web 与接口开发（2 周）

### 目标
- 能独立写一个后端服务

### 技术栈建议
- Gin 和 Gorm
- JSON 接口
- MySQL / PostgreSql（二选一）

### 必须实现
- 基础路由
- 请求 / 响应处理
- CRUD 接口
- 基本错误处理

### 交付物
- 可运行 Web 服务
- API 文档（Markdown 即可）
- 项目结构说明

---

## 七、Phase 3：工具型小项目（2 周）

### 目标
第一次完整工程交付

### 项目示例
- Prompt 管理服务
- 文件管理 / 上传服务
- 简易日志监控工具

### 要求
- 独立设计
- 自己拆解任务
- AI 作为辅助，而不是主脑

### 你需要回答的问题
- 这个项目解决什么问题？
- 哪部分最容易扩展？
- 哪部分你最不确定？

---

## 八、Phase 4：业务型小项目（2–3 周）

### 目标
贴近真实业务的完整交付

### 项目方向
由导师给出业务背景，你需要：
- 理解需求
- 拆解功能
- 自行设计技术方案

### 最终交付
- 可运行项目
- 完整 README
- 简单部署说明
- 项目讲解（口头或文档）

---

## 九、我们对你的最终期望

实习结束时，你应该能够：
- 使用 Go + AI 独立完成项目
- 清楚说明你的设计选择
- 知道如何继续自我成长

如果你能做到这些：
> **你已经具备初级工程师的真实能力，而不仅是“实习经历”。**
